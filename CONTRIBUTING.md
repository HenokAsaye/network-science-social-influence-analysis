# Contributing to Network Science Social Influence Analysis

Thank you for considering contributing to this project! Here are some guidelines to help you get started:

## How to Contribute
- Fork the repository and create your branch from `main` or the latest development branch.
- Make your changes in small, logical commits with clear messages.
- Ensure your code passes all tests and does not break existing functionality.
- Add or update documentation and type hints where appropriate.
- If adding a new feature, include relevant tests and examples.

## Code Style
- Follow PEP8 for Python code.
- Use type hints and docstrings for all functions.
- Use logging instead of print statements for new code.

## Submitting Changes
- Push your branch and open a Pull Request (PR) with a clear description of your changes.
- Reference any related issues in your PR.
- Be responsive to feedback and suggestions from reviewers.

## Community
- Be respectful and constructive in all communications.
- Help others by reviewing PRs and answering questions.

Thank you for helping improve this project!
